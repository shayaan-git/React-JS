Types of Routers

- BrowserRouter
    - History API: pre-post Tabs History
    - URL clean
    - Used in modern apps 
    - Good SEO
    - Drawback: Needs Server Config. (Browser searches for index.html so in server configuration you must have to make index.html)

Hash Router
    - contains ugly URLs - /courses/#/about
    - Used in Older Browsers
    - No SEO

Memory Router
    - React Native

Static Router
    - SSR
